# @isbatak/chakra-ui-mockup-phone

Phone mockup shows a mockup of an iPhone.

## Installation

```sh
pnpm add @isbatak/chakra-ui-mockup-phone
# or
yarn add @isbatak/chakra-ui-mockup-phone
# or
npm i @isbatak/chakra-ui-mockup-phone
```
